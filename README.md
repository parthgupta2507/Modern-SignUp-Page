# 🌟 Modern Authentication UI

A modern and responsive **Sign In / Sign Up** web application built using **HTML5, CSS3, and JavaScript**. This project focuses on creating a clean user interface with smooth animations, interactive form validation, and an enhanced user experience.

# ✨ Features

- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 🔄 Sign In / Sign Up Toggle
- 👁️ Password Visibility Toggle
- 🔐 Password Strength Indicator
- ✅ Real-Time Form Validation
- ⚡ Smooth Button Loading Animation
- 🎉 Success Confirmation Screen
- 🌈 Animated Background Effects
- 💻 Clean and Maintainable Code
